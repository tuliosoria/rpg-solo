# Legacy Reports

These reports document earlier builds of **Varginha: Terminal 1996** and are
preserved here for historical context only. They reference mechanics that have
since been retired (notably `decrypt`, `rewind`, and `present`) and gameplay
balancing that may no longer match the live game.

For current guidance, see:

- `README.md` — player-facing overview and feature list
- `AGENTS.md` and `.github/copilot-instructions.md` — current systems, command
  set, and stale-guidance traps to avoid
- `.github/skills/` — domain-specific guides (game-design, game-content,
  testing, version-bump)

## Index

| File | Original purpose |
|---|---|
| `PLAYTEST_REPORT.md` | Pre-simplification playtest of an earlier build |
| `GAME_EVALUATION.md` | Comprehensive design evaluation against an older feature set |
| `CONSISTENCY_AUDIT_REPORT.md` | Cross-system consistency audit prior to the rewind retirement |

When making changes to the game, prefer the canonical guidance in `AGENTS.md`
over anything in this folder.
